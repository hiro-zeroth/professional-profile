AuthMed

Academic Project | Manila, 2025


View on GitHub


Project Overview


AuthMed is a medicine image classification and authenticity detection system designed to help users verify the authenticity of over-the-counter (OTC) medicines. The system uses deep learning and computer vision to detect and classify medicine images, providing feedback on whether the medicine is genuine or potentially counterfeit. It also supports user feedback and real-time processing to ensure accurate results.



Tech Stack


  • Backend: Python, FastAPI
  • Machine Learning: TensorFlow, Roboflow
  • Object Detection Model: SSD MobileNetV2
  • Frontend/Integration: Base64 and file upload support
  • Tools & IDEs: VS Code, Jupyter Notebook


Key Features


  • Automatic detection and classification of medicines from images
  • Blurriness filtering and out-of-distribution detection for better accuracy
  • Rule-based heuristic system for class decision support
  • Feedback handling mechanism for users to report incorrect predictions
  • FastAPI backend for seamless REST API integration
  • Support for both file uploads and base64 image input


Results


  • Accurate classification of multiple OTC medicines
  • Early detection of potentially counterfeit medicines
  • Real-time feedback for users, improving trust and usability
  • Scalable backend system that supports integration into mobile or web apps
