TASKawaii


Academic Project | Manila, 2025


Download Project



Project Overview


TASKawaii is a student-focused task-management tool designed to improve productivity and streamline the tracking of academic deadlines. It addresses the challenges students face during high-pressure periods by providing an organized platform for managing tasks, deadlines, priorities, and workload. The system allows users to customize task details, gain a clear overview of upcoming responsibilities, and reduce stress through structured planning.



Tech Stack


  • Language:Java
  • Techniques: Serialization to persist user data across sessions
  • Algorithms:Sorting algorithms for task organization
  • Tools: Java runtime environment (JDK), VSCode


Key Features


  • Add, edit, or remove tasks with titles, deadlines, details, and priority levels.
  • Flexible categorization to focus on urgent tasks.
  • Provides a visual overview of upcoming tasks and deadlines.
  • Priority and completed-task classes operate independently from the main task class.
  • Serialization ensures tasks remain saved even after logging out.


Results


TASKawaii has demonstrated significant impact on student productivity by providing structured task organization. It reduces stress during busy periods by giving users clear visibility of deadlines and enhances time management through flexible prioritization and sorting tools. Additionally, the platform ensures reliable data retention through Java serialization and a modular class design, maintaining user input securely across sessions.
