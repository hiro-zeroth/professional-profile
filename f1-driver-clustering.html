F1 Driver Clustering

Personal Project | Manila, 2025


View on GitHub


Project Overview


F1 Driver Clustering is a machine learning project that analyzes the performance of Formula 1 drivers from 1950–2024. Using K-Means clustering and engineered metrics, the project identifies patterns in consistency, dominance, and reliability across different eras of the sport.



Tech Stack


  • Languages: Python
  • Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
  • Tools: Jupyter Notebook, GitHub


Key Features


  • Analyzed driver statistics from multiple decades with era adjustments for fair comparison.
  • Created derived metrics such as adjusted win rate, podium rate, DNF ratio, and qualifying performance.
  • Implemented K-Means clustering and achieved a silhouette score of 0.8.
  • Generated visualizations highlighting cluster groups and driver performance patterns.


Results


The model successfully identified high-dominance drivers, consistent performers, and high-risk drivers based on historical race data. Visual patterns showed that drivers with balanced performance metrics tend to cluster closely, while outliers (dominant champions or highly inconsistent drivers) form distinct groups.
