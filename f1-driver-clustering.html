F1 Driver Clustering

Personal Project | Manila, 2025

View on GitHub

Project Overview

  • Built an F1 Driver Performance Clustering model using Python, Pandas, and scikit-learn to analyze driver performance from 1950–2024.
  • Engineered era-adjusted metrics (win rate, podium rate, DNF ratio, grid position) for fair cross-season comparison.
  • Applied K-Means clustering and achieved a silhouette score of 0.8, indicating strong cluster separation.
  • Visualized results with Matplotlib and Seaborn, deriving insights on driver consistency, dominance, and reliability.
